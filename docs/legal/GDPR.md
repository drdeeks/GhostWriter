# GDPR Compliance

## Data Collection
- We collect only wallet addresses and on-chain interactions for core functionality.
- No personal data (e.g., names, emails) is collected or stored.

## Data Retention
- On-chain data is immutable and retained indefinitely.
- Off-chain data (e.g., IPFS metadata) is retained for 12 months.

## User Rights
- **Right to Access**: Users can request their data via our [Data Request Form](https://ghostwriter.example.com/data-request).
- **Right to Erasure**: Users can burn their NFTs to dissociate from content.
- **Right to Rectification**: Users can update metadata via IPFS.
